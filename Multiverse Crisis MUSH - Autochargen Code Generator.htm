Autochargen Code Generator

This system automatically generates the code for the chargen machine in order to generate a particular character. Note that you must have the proper permissions in order to actually use the machine. Operation is simple - just enter (via cut and paste or manual entry) the values, click the 'Process' button, and copy the 'Result' box below.

Please keep in mind that this does not actually make the character on the MUSH! It merely writes the code to do that for you. You must have a character object on the MUSH with the proper access levels and be in Character Control in order to use this code. Paste the results in the input box in your MUSH client and send it in order to use it, while in Character Control and logged into your chargen-capable character object.
